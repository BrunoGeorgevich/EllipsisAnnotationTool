# Ellipsis Annotation Tool
## Instalando dependências
```python 
pip install -r requirements.txt
```
## Executando app

### Windows

```python 
./run_windows.bat
```

### Linux

```python 
First install qt5 dependencies (rcc)
./run_linux.sh
```

## Compilando App


```python 
pyinstaller main.spec

e depois copia o arquivo main.rcc para a pasta dist/main
```
